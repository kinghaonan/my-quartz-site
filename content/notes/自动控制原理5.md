## 稳定性分析的基本概念

### 稳定性的定义

**稳定性**：系统受到扰动后其运动保持在有限边界的区域内或回复到原平衡状态的性能。

### 分析稳定性的两个基本条件

1. **存在平衡状态**（理论上）
2. **存在扰动**（实际情况）

### 稳定性的三种基本类型

1. **稳定**：系统受到扰动后能够回复到原平衡状态
2. **临界稳定**：系统受到扰动后保持有界运动但不回复
3. **不稳定**：系统受到扰动后运动无界

### 具有稳定性的两种形式

1. **运动有界**：系统响应保持在有限范围内
2. **回复到原平衡状态**：系统最终回到初始平衡点

### 线性系统与非线性系统的稳定性区别

- **非线性系统**：存在多个平衡点，稳定性是相对于特定平衡点的
- **线性系统**：平衡点只有一个，稳定性是全局的

### 渐进稳定性与全局渐进稳定性

1. **渐进稳定性**：当时间趋于无穷大时，受扰运动均回复到原平衡状态
2. **全局渐进稳定性**：对于任意初始扰动引起的受扰运动，系统均是渐进稳定的

**重要结论**：线性系统的稳定即是全局渐进稳定。

### 线性系统稳定性的充要条件

**充要条件**：系统特征方程的根全部具有负实部（传递函数的极点全部具有负实部）。

数学表达：对于特征方程
$$
a_n s^n + a_{n-1} s^{n-1} + \cdots + a_1 s + a_0 = 0
$$
系统稳定的充要条件是所有根$s_i$满足：
$$
\text{Re}(s_i) < 0, \quad i = 1, 2, \ldots, n
$$

**理解**：负实部保证了系统的响应随时间衰减，这是稳定性的数学本质。

## 闭环系统的传递函数

### 单位负反馈系统

对于典型的反馈控制系统：

```
R(s) → ⊕ → [G(s)] → Y(s)
        ↑ (-)      ↓
        └──[H(s)]←─┘
```

当$H(s)=1$时，称为**单位负反馈系统**。

### 闭环传递函数

对于负反馈系统：
$$
\Phi(s) = \frac{Y(s)}{R(s)} = \frac{G(s)}{1 + G(s)H(s)}
$$

对于正反馈系统：
$$
\Phi(s) = \frac{Y(s)}{R(s)} = \frac{G(s)}{1 - G(s)H(s)}
$$

### 特征方程

系统的特征方程为：
$$
1 \pm G(s)H(s) = 0
$$
其中负反馈取"+"，正反馈取"-"。

## 劳斯判据的原理与构建

### 劳斯判据解决的问题

当系统的阶数大于4时，直接求解特征方程的根非常困难。劳斯判据提供了一种代数方法，无需计算根的具体值，仅通过特征方程的系数即可判断根的分布。

### 系统稳定的必要条件

考虑特征多项式：
$$
a_n s^n + a_{n-1} s^{n-1} + \cdots + a_1 s + a_0 = 0
$$
系统稳定的必要条件是：所有系数$a_i > 0$（或同号）。

### 劳斯表的构建步骤

#### Step 01：确定劳斯表的行数
对于n阶系统，劳斯表有n+1行。

#### Step 02：填充前两行
从最高阶项系数开始，按照从上到下、从左到右的顺序填充：
- 第一行：$s^n, s^{n-2}, s^{n-4}, \ldots$
- 第二行：$s^{n-1}, s^{n-3}, s^{n-5}, \ldots$

#### Step 03：计算后续行
从第三行开始，应用规则计算：
$$
b_i = -\frac{1}{a_{i-1,1}} \begin{vmatrix} a_{i-2,1} & a_{i-2,j+1} \\ a_{i-1,1} & a_{i-1,j+1} \end{vmatrix}
$$
其中$a_{i,j}$表示第i行第j列的元素。

**记忆口诀**：固首移尾、负行列式、左下归一。

### 劳斯判据

**劳斯判据**：劳斯表首列极性变换的次数 = 特征方程根位于右半平面的数目。

- **无极性变化**：所有根位于左半平面，系统稳定
- **有极性变化**：有根位于右半平面，系统不稳定
- **极性变化次数**：右半平面根的个数

## 劳斯表的特殊情况和处理方法

### 情况一：首列出现零元素

#### 处理方法1：用极小量ε代替
用一个小正数ε代替0，继续计算劳斯表，然后令ε→0判断符号。

#### 处理方法2：构建对偶多项式
构建对偶多项式$D(s) = s^n D(1/s)$，然后对$D(s)$应用劳斯判据。

### 情况二：某一行全为零

#### 原因分析
某一行全为零表明存在纯偶数阶多项式为原多项式的因式。

#### 处理方法
1. 用上一行构建辅助多项式$A(s)$
2. 对$A(s)$求导得到$A'(s)$
3. 用$A'(s)$的系数代替全零行
4. 继续计算劳斯表

#### 辅助多项式的根分布特点
纯偶数阶多项式的根必然关于原点对称且关于实轴对称：
- 一对实根（在实轴上，互为相反数）
- 一对纯虚根（在虚轴上）
- 两对关于原点对称的复根

### 示例分析

#### 示例 3.1.4：基本劳斯表构建
对于多项式：$s^4 + a_3 s^3 + a_2 s^2 + a_1 s + a_0 = 0$

劳斯表：
| $s^4$ | 1 | $a_2$ | $a_0$ |
|-------|---|---|---|
| $s^3$ | $a_3$ | $a_1$ | 0 |
| $s^2$ | $b_1$ | $b_2$ |  |
| $s^1$ | $c_1$ |  |  |
| $s^0$ | $d_1$ |  |  |

其中：
$$
b_1 = \frac{a_3 a_2 - a_1}{a_3}, \quad b_2 = \frac{a_3 a_0 - 0}{a_3} = a_0
$$
$$
c_1 = \frac{b_1 a_1 - a_3 b_2}{b_1}, \quad d_1 = b_2
$$

#### 示例 3.1.5：劳斯判据应用
对于多项式：$s^3 + 10s^2 + 31s + 1030 = 0$

劳斯表：
| $s^3$ | 1 | 31 |
|-------|---|---|
| $s^2$ | 10 | 1030 |
| $s^1$ | -72 | 0 |
| $s^0$ | 1030 |  |

**分析**：首列符号变化：+ → + → - → +，极性变换两次，说明有2个根在右半平面。

#### 示例 3.1.7：首列出现零的情况
对于多项式：$s^5 + 2s^4 + 3s^3 + 6s^2 + 5s + 3 = 0$

劳斯表（用ε代替0）：
| $s^5$ | 1 | 3 | 5 |
|-------|---|---|---|
| $s^4$ | 2 | 6 | 3 |
| $s^3$ | ε | $\frac{7}{2}$ | 0 |
| $s^2$ | $\frac{6ε - 7}{ε}$ | 3 |  |
| $s^1$ | ... |  |  |
| $s^0$ | ... |  |  |

令ε→0⁺，$\frac{6ε - 7}{ε} → -\infty$，极性变换两次，说明有2个根在右半平面。

## 相对稳定性分析

### 相对稳定性的概念

**相对稳定性**：系统极点向右平移多少，右半平面会出现极点。它量化了系统的"稳定程度"。

### 分析方法

1. **平移变换**：令$s = z - σ$，将原特征方程转换为关于z的方程
2. **应用劳斯判据**：对变换后的方程应用劳斯判据
3. **确定临界值**：找到使系统变为临界稳定的σ值

### 相对稳定性指标

σ值可以作为稳定性的定量指标：
- σ越大，系统相对稳定性越好
- σ=0对应临界稳定
- σ<0表示系统不稳定

### 示例：相对稳定性计算

已知闭环系统传递函数：
$$
\Phi(s) = \frac{1}{s^2 + 5s + 6}
$$

特征方程：$s^2 + 5s + 6 = 0$

令$s = z - 2$，代入得：
$$
(z-2)^2 + 5(z-2) + 6 = z^2 + z = 0
$$

劳斯表：
| $z^2$ | 1 | 0 |
|-------|---|---|
| $z^1$ | 1 | 0 |
| $z^0$ | 0 |  |

**分析**：系统在平移2个单位后变为临界稳定，因此相对稳定性指标σ=2。

**理解**：冲击响应乘以$\exp(2t)$（扩幅）后为临界稳定，说明原系统有2个单位的稳定裕度。

## 劳斯判据在系统设计中的应用

### 参数稳定性分析

对于含参数的系统，劳斯判据可用于确定使系统稳定的参数范围。

#### 设计步骤

1. **确定特征方程**：建立以参数为系数的特征方程
2. **构建参数化劳斯表**：构建包含参数的劳斯表
3. **确定稳定范围**：根据劳斯判据，确定无极性变化的参数范围

#### 示例 3.1.8：参数K的稳定性范围

已知系统特征方程：$s^3 + 3s^2 + (2+K)s + 2K = 0$

劳斯表：
| $s^3$ | 1 | $2+K$ |
|-------|---|-------|
| $s^2$ | 3 | $2K$ |
| $s^1$ | $\frac{3(2+K) - 2K}{3} = \frac{6+K}{3}$ | 0 |
| $s^0$ | $2K$ |  |

稳定条件：
1. 首列全为正：$3 > 0$，$\frac{6+K}{3} > 0$，$2K > 0$
2. 解得：$K > 0$ 且 $K > -6$，即 $K > 0$

#### 示例 3.1.9：单位负反馈系统的K范围

开环传递函数：$G(s) = \frac{K}{s(s+1)(s+5)}$

闭环特征方程：$s^3 + 6s^2 + 5s + K = 0$

劳斯表：
| $s^3$ | 1 | 5 |
|-------|---|---|
| $s^2$ | 6 | K |
| $s^1$ | $\frac{30-K}{6}$ | 0 |
| $s^0$ | K |  |

稳定条件：
1. $\frac{30-K}{6} > 0$ ⇒ $K < 30$
2. $K > 0$
3. 综合得：$0 < K < 30$

## 实践练习

### 练习 3.1.2：劳斯表构建

请根据多项式$s^3 + 3s^2 + 2s + 7 = 0$构建劳斯表。

**解答**：
劳斯表：
| $s^3$ | 1 | 2 |
|-------|---|---|
| $s^2$ | 3 | 7 |
| $s^1$ | $-\frac{1}{3}$ | 0 |
| $s^0$ | 7 |  |

**分析**：首列符号变化：+ → + → - → +，极性变换两次，系统不稳定，有2个根在右半平面。

### 练习 3.1.3：参数稳定性分析

已知系统特征方程：$s^3 + 3s^2 + 2s + k = 0$，求使系统稳定的k取值范围。

**解答**：
劳斯表：
| $s^3$ | 1 | 2 |
|-------|---|---|
| $s^2$ | 3 | k |
| $s^1$ | $\frac{6-k}{3}$ | 0 |
| $s^0$ | k |  |

稳定条件：
1. 首列全为正：$3 > 0$，$\frac{6-k}{3} > 0$，$k > 0$
2. 解得：$k > 0$ 且 $k < 6$
3. 综合得：$0 < k < 6$

**理解**：当$k$在0到6之间时，系统稳定；$k=0$时系统临界稳定；$k<0$或$k>6$时系统不稳定。

### 练习 3.1.4：特殊情况的处理

已知系统特征方程：$s^5 + 7s^4 + 6s^3 + 42s^2 + 8s + 56 = 0$，判断系统稳定性。

**解答**：
劳斯表：
| $s^5$ | 1 | 6 | 8 |
|-------|---|---|---|
| $s^4$ | 7 | 42 | 56 |
| $s^3$ | 0 | 0 | 0 |

出现全零行，构建辅助多项式：
$A(s) = 7s^4 + 42s^2 + 56 = 7(s^4 + 6s^2 + 8)$
$A'(s) = 28s^3 + 84s = 28(s^3 + 3s)$

用$A'(s)$系数代替全零行：
| $s^5$ | 1 | 6 | 8 |
|-------|---|---|---|
| $s^4$ | 7 | 42 | 56 |
| $s^3$ | 28 | 84 | 0 |
| $s^2$ | 21 | 56 |  |
| $s^1$ | $\frac{28}{3}$ | 0 |  |
| $s^0$ | 56 |  |  |

**分析**：首列全为正，无极性变化，但存在全零行表明有纯虚根。辅助多项式$A(s)=0$的根为$s^2 = -2, -4$，即$s = \pm j\sqrt{2}, \pm j2$。系统有4个纯虚根和1个左半平面实根，属于临界稳定。

### 练习 3.1.5：相对稳定性分析

已知系统传递函数：$\Phi(s) = \frac{1}{s^3 + 4s^2 + 6s + 4}$，求系统的相对稳定性指标σ。

**解答**：
特征方程：$s^3 + 4s^2 + 6s + 4 = 0$

令$s = z - σ$，代入得：
$(z-σ)^3 + 4(z-σ)^2 + 6(z-σ) + 4 = 0$

展开并整理：
$z^3 + (4-3σ)z^2 + (6-8σ+3σ^2)z + (4-6σ+4σ^2-σ^3) = 0$

应用劳斯判据，求使系统临界稳定的σ值：
劳斯表：
| $z^3$ | 1 | $6-8σ+3σ^2$ |
|-------|---|------------|
| $z^2$ | $4-3σ$ | $4-6σ+4σ^2-σ^3$ |
| $z^1$ | $b_1$ | 0 |
| $z^0$ | $c_1$ |  |

其中：
$b_1 = \frac{(4-3σ)(6-8σ+3σ^2) - (4-6σ+4σ^2-σ^3)}{4-3σ}$

令$b_1 = 0$，解得$σ ≈ 0.59$

**结论**：相对稳定性指标σ=0.59，系统有0.59个单位的稳定裕度。

## 小结与总结

### 核心概念总结

1. **稳定性定义**：系统受到扰动后其运动保持在有限边界区域内或回复到原平衡状态的性能。

2. **稳定性类型**：
   - 稳定：回复到原平衡状态
   - 临界稳定：保持有界运动但不回复
   - 不稳定：运动无界

3. **线性系统稳定性充要条件**：特征方程的所有根具有负实部（传递函数的所有极点位于左半平面）。

4. **劳斯判据**：基于特征方程系数的代数稳定性判据，无需计算根的具体值。

### 重要方法总结

#### 劳斯表构建方法
1. **确定行数**：n阶系统→n+1行
2. **填充前两行**：按幂次交错填充系数
3. **计算后续行**：应用"固首移尾、负行列式、左下归一"规则

#### 特殊情况处理
1. **首列出现零**：
   - 方法1：用极小量ε代替，令ε→0判断
   - 方法2：构建对偶多项式
2. **某行全为零**：
   - 构建辅助多项式$A(s)$
   - 对$A(s)$求导得$A'(s)$
   - 用$A'(s)$系数代替全零行

#### 参数稳定性分析
1. 建立含参数的特征方程
2. 构建参数化劳斯表
3. 根据首列符号确定参数稳定范围

#### 相对稳定性分析
1. 令$s = z - σ$进行平移变换
2. 对变换后的方程应用劳斯判据
3. 确定使系统临界稳定的σ值

### 关键公式回顾

1. **闭环传递函数**（负反馈）：
   $$
   \Phi(s) = \frac{G(s)}{1 + G(s)H(s)}
   $$

2. **特征方程**：
   $$
   1 + G(s)H(s) = 0 \quad \text{(负反馈)}
   $$

3. **劳斯表元素计算**：
   $$
   b_i = -\frac{1}{a_{i-1,1}} \begin{vmatrix} a_{i-2,1} & a_{i-2,j+1} \\ a_{i-1,1} & a_{i-1,j+1} \end{vmatrix}
   $$

4. **相对稳定性变换**：
   $$
   s = z - σ
   $$
